# 帮助中心导航与搜索

帮助中心首页优先提供两个入口：

- 搜索框：适合直接输入问题或任务
- 目录导航：适合按模块逐级浏览

## 搜索体验设计

Phase 1 的前端入口已经准备好，目标搜索体验包括：

- 拼写容错
- 同义词与关键词召回
- 支持自然语言输入
- 按语言、版本、产品线过滤

后续会逐步升级为 `hybrid search`，让“怎么撤回流程”这类表达也能命中文档。

## 阅读体验设计

文档页会统一展示：

- 所属目录
- 最近更新时间
- 适用版本
- 编辑入口

对于长文档，会补充：

- 右侧目录锚点
- 关联文档
- 常见问题跳转
